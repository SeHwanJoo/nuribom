   2) 메인페이지 생성시 네트워크 통신(추천 관광지, 코스)

   2) 관광지/코스 분류 검색

- 여행지(관광지)
1. 메인 + 리스트뷰
   1) 메인페이지 생성시 네트워킹 → 리스폰스로 초기화면 구성
   2) 지역 선택 + 필터링 → 네트워킹 리스폰스로 리스트뷰 구성
   3) 지역 선택 + 키워드 검색 → 네트워킹 리스폰스로 리스트뷰 구성
   4) 시설 필터 → 서버에서 보내준 모든 값들을 안드단에서 구분
   5) 여행지에서는 인기순만 존재(최신순 삭제)
   6) 찜하기 버튼

2. 여행지 상세보기 + 뷰페이져 
   1) 페이지 생성시 네트워킹
   2) 탭 선택시 포지션 이동 → 일단 탭+뷰페이져로 대체, 추후 수정 예정
   3) 길안내 버튼 → 지도 페이지로 이동 → T-map이용 길안내 시작
